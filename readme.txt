Plugin Floating Ads WordPress - WP Simple Floating Ads by Masmon adalah plugin untuk menampilkan ads (iklan) yang posisinya fixed.

Dengan menggunakan plugin WP Simple Floating Ads by Masmin ini Anda bisa menampilkan iklan dengan berbagai opsi iklan seperti single dan multiple Ads.

Tiap-tiap opsi memiliki 3 opsi jenis iklan yaitu: text, image dan adsense Google. Tujuannya adalah untuk memberikan fleksibelitas yang tinggi untuk mendongkrak jumlah raihan iklan dari blog Anda.

Fitur Simple Floating Ads by Masmon

Plugin ini memiliki sejumlah fitur seperti:

- 2 opsi pilihan ads: single dan multiple
- Masing-masing 3 opsi pilihan text ads, image ads dan adsense ads
- Multiple ads ditampilkan secara random
- Text ads menampilkan head, conten dan button ads
- Image ads menampilkan banner with kink dan button
- Adsense ads menampilkan Google adsense
- 3 plihan lokasi penempatan
- dll

==Instalasi==

Plugin ini membutuhkan plugin Carbon Fields versi V3.6.0 untuk di install. (wajib) untuk mendownload plugin tersebut terlebih dahulu. Untuk instalasi ikuti petunjuk berikut:

- Jika WordPress Anda belum terinstall plugin Carbon Fields silahkan untuk mendownloadnya disini
- Install dan Aktifkan Carbon Fields
- Download plugin WP Fake SalesX by Masmon -> install -> aktifkan
- Isi konfigurasi yang ada pada menu setting -> FakeSalesX options sesuai selera dan kebutuhan Anda
